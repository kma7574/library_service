# 도서관 대출 서비스(BackEnd Project with FLASK)

  
  

## 프로젝트 소개

  

##### 도서관 대출 서비스는 레이서 도서관에 있는 책을 온라인으로 관리할 수 있는 웹 서비스입니다.

  

##### 책의 상세 정보를 확인할 수 있는 페이지와 대여/반납 기능을 통해 도서 관리를 할 수 있는 웹 서비스를 제작하게 됩니다.

  ------------------------------------------------
  

## 주요 사용 기술


- Flask

- SQLAlchemy

- PyMySQL

- MySQL

- JQuery

- HTML with Flask Jinja2

------------------------------------------------
